# github-search
python编写的爬虫，用于爬取github上搜索中的内容，分为scan.py 和download.py俩部分

scan.py可以实现对搜索内容的爬取，并且保存信息到data.json文件中
在图中scan.py添加apiaccess_token的值

load.py可以实现对data.json文件中所保存的仓库进行下载并创建且保存到download.zip文件中
